## 🚀 Quick Deploy

### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files (or push via git)
3. Go to Settings → Pages
4. Set source to "main" branch, root folder
5. Your site will be live at `https://yourusername.github.io/repo-name`

### Cloudflare Pages
1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Click "Create a project"
3. Either connect your GitHub repo or drag-and-drop the entire folder
4. Your site will be live instantly with a cloudflare.dev URL

### Netlify
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag the entire folder into the browser
3. Your site will be live instantly with a netlify.app URL

## 🌐 Custom Domain (captainmax-li.com)

After deploying to any platform above:

1. Get your platform's URL (e.g., `captainmax.pages.dev`)
2. Log into GoDaddy DNS settings
3. Update the CNAME record:
   - Type: CNAME
   - Name: www
   - Value: [your platform URL]
4. Add an A record for the root domain (check your platform's docs for the IP)

All three platforms provide free SSL certificates automatically.
